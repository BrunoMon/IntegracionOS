Volver

¿Como veo el estado mis autorizaciones?

Selecciona el punto de menu "Presentar Facturas".
Ahí podras visualizar la lista de tus autorizaciones filtradas por período y estado.
Pudes modificar el filtro introduciendo un nuevo período (yyyymm) y/o nuevo estado de la lista desplegable.
Presiona el boton "Consultar Autorizaciones".
La lista se actualizará mostrando los datos que coincidan con la selección.
En caso de no encontrarse autorizaciones que cumplan las condiciones, se mostrara una lista vacía.
Volver

¿Como genero autorizaciones para el periodo actual/próximo?

Selecciona el punto de menu "Autorizaciones".
Presiona el boton "Generar autorizaciones para el período actual".
Luego de unos segundos se desplegara una ventana presentando un detalle de las autorizaciones creadas y/o rechazadas.
Puedes presionar el boton de "Consultar autorizaciones" para seguir consultando el estado de tus autorizaciones.
Si no hubiere prestaciones pendientes de autorización, se mostrará una lista vacía.
Sigue los mismos pasos para generar autorizaciones para el próximo período presionando el boton "Generar autorizaciones para el período próximo."
Volver

¿Como consulto mis expedientes?

Selecciona el punto de menu "Autorizaciones".
Selecciona de la lista desplegable el año por el cual deseas filtrar tus expedientes.
Se mostrará automaticamente una lista de beneficiarios que tengan expedientes para ese período.
Si no hubiere expedientes para ese período se mostrará una lista vacía.
Presionando el boton en la parte superior izquierda de cada tarjeta de beneficiario podras desplegar/ocultar la lista de expedientes activos para ese beneficiario.

¿Como llego a la presentación de una factura?

El sistema cuenta con dos formas de acceder a la presentación de una factura.
Puedes usar la que más te convenga de acuerdo a tu flujo de trabajo.
Opción de menu "Autorizaciones".
Simplemente selecciona el período que desees facturar, busca el beneficiario en la lista que aparece automaticamente.
Si no hubiere expedientes para ese período se mostrará una lista vacía.
Presionando el boton en la parte superior izquierda de cada tarjeta de beneficiario podras desplegar/ocultar la lista de expedientes activos para ese beneficiario.
En parte derecha de cada tarjeta de expediente aparecerá un boton "Presentar factura".
Si el expediente no tuviere autorizaciones pendientes de facturación, el boton "Presentar factura" no se mostrará.
Opción de menu "Presentar facturas".
Selecciona el período (yyyymm) y el estado APROBADO ó RECHAZADO de la lista desplegable.
Presiona el boton "Consultar autorizaciones".
Se desplegará una lista de autorizaciones y un boton "Presentar factura" en cada tarjeta que se encuentre habilitada para hacer la presentación.
Las autorizaciones que ya tengan factura presentada no presentarán el boton "Presentar factura".
Si la factura fue presentada y rechazada por la obra social, la autorización volverá a presentar el boton "Presentar factura".

¿Como conformo y presento una factura?

Paso 1: Elegí el período autorizado.
Selecciona el período la lista desplegable.
Presiona el boton "Continuar".
Paso 2: Ingresá los datos de la factura.
Introduce los datos en el formulario.
El sistema permitirá introducir los datos libremente pero al momento de continuar, estos seran validados y no se podrá continuar hasta que todos los datos pasen la validación correspondiente.
Presiona el boton "Continuar".
Observa los mensajes de error en la validación y corrige si fuera necesario.
Paso 3: Subí las imágenes correspondintes.
En este paso deberas subir los archivos correspondientes a las imágenes de la factura,ddjj y certificado de asistencia.
Presiona el boton que corresponda de acuerdoo a la imagen que quieras presentar. Recuerda que el limite máximo para cada uno de estos archivos es de 3 mega bytes.
Presiona el boton "Continuar".
Observa los mensajes de error en la validación y corrige si fuera necesario.
Paso 4: Verificá los datos y presentá.
En este paso te presentamos todos los datos conformados para que sean verificados por ultima vez antes de su presentación.
Si bien la factura ya se encuentra conformada y guardada. (puedes continuar con la carga en otro momento), todavía no ha sido presentada a la Obra Social.
Presiona el boton "Presentar".

¿Como hago el seguimiento de las facturas presentadas?

La obra social enviará un mail al prestador por cada cambio de estado que se produzca en una de sus facturas.
De todas formas para consultar el estado de las facturas presentadas presiona el punto de menu "Consultar Facturas".
Introduce el período (yyyymm) y selecciona el estado de la lista desplegable.
Presiona el boton "Consultar Facturas".
El sistema desplegará una lista con las facturas que cumplan el criterio seleccionado.
Pudes observar los datos y el estado de cada factura.
También puedes controlar los archivos de imágenes presentados haciendo click en el nombre del documento correspondiente.
